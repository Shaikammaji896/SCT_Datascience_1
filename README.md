# SkillCraft Technology - Data Visualization Task

## Task Objective
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable using Microsoft Excel.

---

## Project Description
This project demonstrates data visualization techniques using sample datasets in MS Excel.

Two visualizations were created:

1. Histogram for Age Distribution
2. Bar Chart for Gender Distribution

These charts help in understanding how data is distributed within a population.

---

## Tools Used
- Microsoft Excel
- GitHub

## Dataset Used

### Age Dataset
Sample age values were used to create a histogram showing age distribution.

### Gender Dataset
Sample gender values were used to create a bar chart showing gender distribution.
